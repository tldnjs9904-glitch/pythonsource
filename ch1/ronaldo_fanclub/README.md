# SIUUU CLUB

Python 표준 라이브러리만 사용한 호날두 팬클럽 학습용 웹사이트입니다.

## 실행

```powershell
cd ch1/ronaldo_fanclub
python app.py
```

브라우저에서 `http://localhost:8000`을 여세요.

- HTML: 페이지의 뼈대와 내용
- CSS: 색상, 배치, 반응형 디자인
- JavaScript: 폼 입력 및 서버 통신
- Python: 회원가입, 로그인, 비밀번호 해시, SQLite 저장

실서비스에서는 HTTPS, 영구 세션 저장소, CSRF 보호, 속도 제한 등을 추가해야 합니다.
